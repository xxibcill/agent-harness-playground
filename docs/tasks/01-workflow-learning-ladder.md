# 01 Workflow Learning Ladder

Status: Done

## Goal

Define the intended educational progression for workflows before adding new implementations.

## Scope

- Confirm the workflow names and ordering for the learning ladder.
- Keep the progression incremental so each workflow adds only one new idea.
- Treat `anthropic.respond` as the first provider-backed workflow, not the first advanced logic
  workflow.

## Deliverables

- A documented workflow sequence for the repository.
- Short teaching notes for what each workflow demonstrates.
- A clear statement of which workflow remains the simplest smoke test.

## Exit Criteria

- The team can point to one canonical workflow order.
- Each workflow in the sequence has exactly one primary teaching concept.
- The plan explicitly states that this repository is for education.

## Notes

- Do not implement runtime code in this task.
- If naming feels unclear, prefer names that reveal the learning step directly.
